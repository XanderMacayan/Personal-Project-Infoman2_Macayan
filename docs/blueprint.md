# **App Name**: NEU Library Log

## Core Features:

- Visitor Entry & Identification: Allow visitors to log their entry via NEU School ID (RFID tap) or institutional email. Display their name, college/office upon successful validation, along with a 'Welcome to NEU Library!' message.
- Purpose of Visit Selection: Provide a pre-selected list of options for visitors to specify their purpose of visit (e.g., reading books, thesis research, computer use, assignments) upon entry.
- Admin Dashboard: Display key visitor statistics (per day, week, month, or custom duration) in an intuitive dashboard format with individual cards.
- Admin Visitor Blocking: Enable administrators to block or unblock specific visitor entries through the application.
- Statistical Filtering: Allow admins to filter visitor statistics based on reason for visiting, college, and visitor type (employee/non-employee).
- PDF Report Generation: Generate and allow download of PDF reports for selected visitor statistics.
- AI-powered Visitor Trend Summary Tool: Provide an AI tool for administrators that automatically summarizes visitor trends and highlights unusual patterns or significant changes in library usage based on collected data.

## Style Guidelines:

- Primary color: A sophisticated and trustworthy blue (#306FE8), evoking knowledge and professionalism for a university library setting.
- Background color: A very light, desaturated blue-grey (#F0F3F5) to create a clean, calm, and focused atmosphere, subtly complementing the primary blue.
- Accent color: A vibrant yet calm cyan (#6CE7E6), offering excellent contrast for interactive elements, calls to action, and highlighting important data.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and highly legible appearance, suitable for displaying data and text efficiently across the application.
- Utilize a consistent set of clean, line-style icons to ensure clarity and professionalism, enhancing the functional aspect of the app without distraction.
- Employ a structured and clear layout, prioritizing readability for visitor input screens and intuitive data presentation for the administrative dashboard.
- Implement subtle and functional animations, such as smooth transitions for screen changes and immediate feedback for visitor entries or data updates, enhancing user experience without being intrusive.
